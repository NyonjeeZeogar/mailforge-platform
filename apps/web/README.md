# MailForge Platform

MailForge is an email hosting and delivery platform built for small businesses that need affordable, reliable custom domain email.

## Features (Planned)

* Custom domain email hosting
* Mailbox + alias management
* Webmail interface
* SMTP + IMAP support
* Admin dashboard
* Domain + DNS automation
* Multi-tenant architecture

## Current Status

Initial UI imported from Vite-based dashboard.
Migration to Next.js + TypeScript in progress.

## Project Structure

* `apps/vite-ui` → legacy dashboard UI
* `apps/web` → upcoming Next.js application

## Roadmap

1. Import and stabilize UI
2. Migrate to Next.js
3. Add backend (PostgreSQL + API)
4. Integrate mail infrastructure
5. Launch MVP
